## Responsive notes

* All sections use **fluid containers** and Tailwind breakpoints (`md:`, `lg:`). The hero scales via heading tokens (hd‑2/hd‑5); cards stack on mobile.
* Replace placeholder blocks with your **Figma exports** or SVGs; we keep the layout responsive by avoiding fixed widths where possible.
